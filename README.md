Protect sensitive information while recording screens, demos, or tutorials.

**Screen Blur** is a lightweight Chrome extension that lets you blur any element on a webpage with a click. Ideal for hiding API keys, credentials, personal data, or confidential UI during screen recordings.

---

## Features

- Click any element to blur or unblur  
- Toggle blur mode on/off  
- Clear all blurs instantly  
- Visual hover feedback  
- Privacy-first: runs entirely locally  
- Lightweight, no external dependencies  

---

## Use Cases

- Hide API keys or tokens in code editors  
- Blur credentials in forms or dashboards  
- Conceal sensitive metrics or personal data  
- Screen recordings, demos, tutorials, presentations
